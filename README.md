# User Management Backend (Spring Boot + JWT)

A production-ready user authentication and management REST API built using Spring Boot, Spring Security, and JWT.

## Features
- User Registration
- User Login (JWT-based authentication)
- Role-based Access Control
- Token Validation & Refresh
- MySQL / PostgreSQL Support
- Clean Layered Architecture

## Tech Stack
- Java 17
- Spring Boot
- Spring Security
- JWT
- Maven
- MySQL / PostgreSQL

## How to Run
mvn clean install
mvn spring-boot:run

# User Management Backend

A Spring Boot backend application for managing users with RESTful APIs and MySQL database integration.

## 🚀 Technologies Used

* Java
* Spring Boot
* Spring Data JPA
* MySQL
* REST API
* Maven

## ✨ Features

* Create Users
* View User Details
* Update User Information
* Delete Users
* Database Integration
* RESTful API Endpoints
* Exception Handling

## 📌 Purpose

This project was developed to improve backend development skills using Java and Spring Boot.

## 🛠️ Setup Instructions

1. Clone the repository
2. Configure MySQL database
3. Update application.properties
4. Run using:
   mvn spring-boot:run
