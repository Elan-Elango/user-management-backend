# User Management Backend

A Spring Boot backend application for managing users with RESTful APIs and MySQL database integration.

## 🚀 Technologies Used

* Java
* Spring Boot
* Spring Data JPA
* MySQL
* REST API
* Maven

## ✨ Features

* Create Users
* View User Details
* Update User Information
* Delete Users
* Database Integration
* RESTful API Endpoints
* Exception Handling

## 📌 Purpose

This project was developed to improve backend development skills using Java and Spring Boot.

## 🛠️ Setup Instructions

1. Clone the repository
2. Configure MySQL database
3. Update application.properties
4. Run using:
   mvn spring-boot:run
